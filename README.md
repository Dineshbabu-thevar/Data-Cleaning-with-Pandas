# 🧹 Data Cleaning with Pandas

## 📌 Project Overview

This project demonstrates a complete data cleaning workflow using Python and Pandas. A messy student dataset was cleaned by handling missing values, removing duplicate records, standardizing text formatting, and exporting a clean dataset.

---

## 🛠 Technologies Used

- Python 3
- Pandas

---

## 📂 Project Structure

```
Data_Cleaning_Project
│
├── data
│   ├── messy_students.csv
│   └── cleaned_students.csv
│
├── main.py
└── README.md
```

---

## 📋 Data Cleaning Steps

- Read CSV dataset
- Handle missing values using mean
- Remove duplicate rows
- Remove extra spaces
- Standardize city names
- Reset row index
- Save cleaned dataset

---

## 📊 Results

- Missing Age values filled
- Missing Marks values filled
- Duplicate records removed
- City names standardized
- Clean dataset exported successfully
